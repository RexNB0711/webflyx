# Webflyx

## Project Overview

Webflyx is a command-line content management application developed as part of the Boot.dev Python curriculum. The project simulates a lightweight content management system (CMS) that allows users to organize, manage, and update website content through a structured interface.

The application works with markdown and CSV files to store and manage information such as website content, titles, and quotes. Through this project, I gained hands-on experience with Python programming, file handling, data organization, and command-line application development.

## Features

* Manage website content through the command line
* Read and update markdown files
* Store and retrieve data from CSV files
* Organize content into categories and collections
* Add and manage quote databases
* Process and update website titles and content
* Practice working with structured file systems
